# Polar
Polar is Polibatam guide tour with augmented reality base Unity + Vuforia

## Feature
- Marker
- 3D Layout Landscape
- 3D Room
- Maps (use MapBox package)

## Usage
- Download Marker and print
- Open apps and scan marker
- Explore apps
